1. Про сервіс

Fistserious — це платформа для автоматизованого аналізу достовірності новин. Сервіс застосовує алгоритми лінгвістичного аналізу, крос-перевірки джерел і факт-чекінгу для оцінки матеріалів на предмет ознак маніпуляції, дезінформації та клікбейту.

Сервіс розроблено як навчальний проєкт і не є комерційним продуктом. Результати аналізу носять виключно інформаційний характер і не можуть використовуватись як підстава для юридичних, фінансових або будь-яких інших рішень.

2. Умови використання

Використовуючи сервіс, ви погоджуєтесь:

  • Надавати достовірну інформацію при реєстрації (дійсний email, реальний username).
  • Не передавати доступ до облікового запису третім особам.
  • Використовувати результати аналізу відповідально, розуміючи їх алгоритмічну природу.
  • Не намагатись обійти механізми безпеки або навантажити сервіс штучним трафіком.
  • Дотримуватись чинного законодавства України при роботі з контентом.

3. Заборонений контент

Забороняється додавати до сервісу матеріали, що:

  • Містять заклики до насилля, ненависті або дискримінації.
  • Порушують авторські права третіх осіб.
  • Є свідомою дезінформацією, що завдає шкоди особам або організаціям.
  • Містять персональні дані третіх осіб без їхньої згоди.
  • Порушують законодавство України або міжнародні норми права.

4. Відмова від відповідальності

⚠ Алгоритмічний аналіз не замінює професійну журналістську верифікацію. Оцінки достовірності є автоматизованими і можуть містити похибки. Fistserious не несе відповідальності за рішення, прийняті на підставі результатів аналізу.

  • Сервіс надається «як є» без гарантій безперервної роботи або абсолютної точності.
  • Алгоритми оптимізовані для англійської та української мов; точність для інших мов знижена.
  • Зовнішні API (NewsAPI, Guardian, ClaimBuster тощо) можуть бути недоступні — це впливає на окремі шари аналізу.
  • Адміністрація залишає за собою право змінювати алгоритми без попереднього повідомлення.

5. Облікові записи

Обліковий запис може бути заблокований або видалений у разі порушення цих умов. Ви маєте право в будь-який момент видалити свій акаунт і всі пов'язані дані, звернувшись до адміністрації.

Паролі зберігаються у хешованому вигляді. Адміністрація не має доступу до вашого пароля у відкритому вигляді.

6. Політика конфіденційності

Fistserious збирає мінімально необхідний обсяг даних для роботи сервісу. Ми не передаємо ваші особисті дані третім особам і не використовуємо їх у рекламних цілях.

Які дані ми збираємо

  • Email — для автентифікації та відновлення доступу.
  • Username — публічне ім'я у системі.
  • Результати аналізу — зберігаються для відображення в особистому кабінеті.
  • Додані новини — матеріали, що ви завантажуєте вручну.
  • Обрані новини (favorites) — список статей, відмічених вами.

Чого ми не збираємо

  • Ми не збираємо cookies для відстеження або рекламних цілей.
  • Ми не використовуємо сторонні трекери (Google Analytics тощо).
  • Ми не зберігаємо IP-адреси користувачів у прив'язці до профілю.

7. Дані та їх обробка

Зберігання

Дані зберігаються в базі MongoDB на сервері проєкту. Доступ до бази обмежений і захищений паролем.

Термін зберігання

Дані зберігаються протягом усього часу існування вашого облікового запису. Після видалення акаунту всі пов'язані дані видаляються протягом 30 днів.

Треті сторони

Для аналізу сервіс звертається до зовнішніх API (NewsAPI, The Guardian, GNews, Wikipedia). Ці запити містять лише пошукові ключові слова з заголовків статей — без ваших персональних даних.

8. Контакти

З питань конфіденційності, видалення акаунту або порушень умов використання звертайтесь до адміністрації через розділ «Адмін-панель» або напряму за адресою проєкту.

Ці умови можуть оновлюватись. Актуальна версія завжди доступна за посиланням /terms.
